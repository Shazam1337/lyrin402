# Quick Start Guide

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Основные функции

### Навигация
- **Hub** (/) - Главная страница с KPI, графиками и Live Ticker
- **Feed** (/feed) - Лента событий (волн)
- **Echo Core** (/echo) - Баланс и статистика
- **Resonance Map** (/map) - Интерактивный граф сети
- **Transmit** (/transmit) - Форма отправки волны
- **Manifest** (/manifest) - Манифест с эффектом печати

### Управление симуляцией
- Нажмите `` ` `` (backtick) или `Ctrl+Esc` для открытия панели управления
- Доступные скорости: Slow, Normal, Fast, Hype
- Кнопки: Inject 50 Waves, Trigger Spike, Reset Seeds

### Подключение кошелька
- Нажмите "Connect Resonance" в навбаре
- Выберите Phantom или Solflare
- Кошелёк используется только для чтения адреса (без ончейн-операций)

## Особенности

- Все данные симулируются локально
- Симулятор работает постоянно, даже когда страница не открыта
- Данные сохраняются в localStorage
- Все анимации и эффекты работают в реальном времени

## Сборка для продакшена

```bash
npm run build
npm start
```
